# Partoska Laboratory Scoop Bucket

Official [Scoop](https://scoop.sh) bucket for [Partoska Laboratory](https://lab.partoska.com) tools.

## Install

```powershell
scoop bucket add partoska https://github.com/partoska/scoop-bucket
```

## Apps

| App | Description |
|-----|-------------|
| `p6a` | Command-line tool to manage and sync your event photos from Partoska |

### p6a

```powershell
scoop install partoska/p6a
```
